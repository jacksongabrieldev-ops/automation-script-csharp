# 📝Automação de Relatório de Pagamentos

Script de automação desenvolvido em C# para análise de dados financeiros.

## 📌 Funcionalidades

- Leitura de arquivo CSV
- Identificação de pagamentos pendentes
- Geração automática de relatório
- Cálculo do total de valores pendentes

## 🛠 Tecnologias utilizadas

- C#
- .NET Framework

## 📂 Funcionamento

O sistema lê um arquivo CSV contendo:

- Nome do cliente
- Email
- Valor pendente

Após a leitura dos dados, o programa gera um relatório contendo:

- Lista de clientes com pagamentos pendentes
- Total de valores pendentes
- Quantidade de clientes com dívida

## 🎯 Objetivo do projeto

Praticar automação de tarefas e manipulação de arquivos utilizando C#.

## 🚀 Aprendizados

- Automação de processos
- Leitura de arquivos CSV
- Geração de relatórios automatizados
